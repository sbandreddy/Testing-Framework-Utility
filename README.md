**Project Overview:**

This project is a compact utility that converts low level trade data into two structured Excel Workbooks used to drive an automated testing framework.

**Need:** Workbooks had to be manually configured and it used to take up to 1-2 weeks per client (depending on client data).

**Outcome:** Eliminated repetitive manual labour, drastically improved output quality of the testing framework.

**Note:** Due to Company's Proprietary polices, all Attributes and Naming Conventions used are synthetic.
